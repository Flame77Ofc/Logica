Operadores de Comparação
Os Operadores de Comparação comparam números com os operadores:
    >       Maior que
    <       Menor que
    >=      Maior ou igual que
    <=      Menor ou igual que
    !=      Diferente de
    ==      Igual a
O resultado disso retorna um valor booleano ou seja, true ou false

Algoritmo "Operadores_Comparacao"
Início
numero1 = 5
numero2 = 10


//Comparando
numero1 > numero2 //    false
numero1 < numero2 //    true
numero1 >= numero2 //   false
numero1 <= numero2 //   true
numero1 != numero2 //   true
numero1 == numero2 //   false
Fim
